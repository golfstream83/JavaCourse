Описание класов

1) Item - класс, описывающий модель заявки. В данном классе с помощью геттеров и сеттеров получаем или задаём значения параметров заявки.
2) Tracker - класс, использующийся в качестве хранилища заявок. Содержит методы добавления, редактирования, удаления заявки, отображения всех заявок, отображения отфильтрованных заявок, добавления комментария.
3) MenuTracker - класс управляющий хранилищем заявок. С помощью данного класса формируется меню программы.
4) AddItem - внутренний класс MenuTracker. Нужен для принятия введённых пользователем значений параметров Имя и Описание и вызова метода добавления новой заявки класса Tracker.
5) EditItem - внутренний класс MenuTracker. Нужен для принятия введённых пользователем значений параметров ID, Имя, Описание и вызова метода редактирования заявки класса Tracker.
6) ShowItems - внутренний класс MenuTracker. Нужен для отображения в консоли всех заявок.
7) DeleteItem - внутренний класс MenuTracker. Нужен для принятия введённого пользователем значения параметра ID и вызова метода удаления заявки класса Tracker.
8) ShowItemsFilteredByName - внутренний класс MenuTracker. Нужен для принятия введённого пользователем значения строкового параметра и вызова метода класса Tracker, который отображает все заявки содержащие значение данного параметра.
9) AddComment - внутренний класс MenuTracker. Нужен для принятия введённого пользователем значения параметра ID и вызова метода класса Tracker для добавления комментария к заявке .
10) ExitProgram - внутренний класс MenuTracker. Нужен для выхода из программы.
11) StartUI - класс для запуска программы.
12) ConsoleInput - этот класс нужен для считывания введённых пользователем в консоли данных и передачи их с помощью метода.
13) StubInput - этот класс нужен для тестирования кода. Он содержит в себе статические значения параметров и передаёт их, эмулируя ввод значений в консоли пользователем.
14) StartUITest - этот класс нужен для тестирования кода. Принимает в качестве значений параметров данные из класса StubInput.
